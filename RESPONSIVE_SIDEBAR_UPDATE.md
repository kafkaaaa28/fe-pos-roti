# Responsive Sidebar Update

Perubahan pada update ini:

1. Navbar atas disembunyikan untuk role internal yang sudah memiliki sidebar: Manager, Staff, dan Kasir.
2. Navbar tetap tampil untuk public/customer karena customer belum memakai sidebar internal.
3. Sidebar desktop dibuat sticky penuh setinggi layar.
4. Pada tablet/mobile, sidebar berubah menjadi drawer dari sisi kiri dengan tombol Menu mengambang.
5. Konten halaman manager, staff, dan kasir diberi padding responsif agar tidak tertutup tombol menu.
6. Tabel dibuat horizontal scroll pada layar kecil.
7. Modal dibuat lebih aman untuk layar HP dengan max-height dan scroll internal.
